# Syntexchub Port Scanner

## Description
A simple port scanner that checks open ports on a target system.

## Features
- Scan open ports
- Fast and lightweight
- Easy to use

## Installation

```bash
git clone https://github.com/Surbhi992/Syntexchub_PortScanner.git
cd Syntexchub_PortScanner
```

## Usage
```bash
python port_scanner.py
```
